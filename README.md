# lorem-photography

## Description
The lorem-photography web application is made to imitate a photography company's web page.

## Features
* A user can click view more photos and more photos will display using the Unsplash API .
* The user reviews on the welcome page are genreated from a API on the server.
* The navbar tabs will bring the user to the corresponding sections.
* The icons in the footer will take the user to my social pages
* The contact hyperlink will open up a email to contact the photography company. 

## Technology used
* HTML/CSS/Javascript
* Unsplash API
* Review API (buit in the server)
